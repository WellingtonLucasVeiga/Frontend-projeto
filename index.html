O que é ser Front end?

Um programador front-end constrói a interface visual dos Websites, portanto ele cuida da parte visível do site, por isso o nome de Front.

Diferença entre programador e desenvolvedor

Um programador está diretamente ligado à produção e elaboração do código, se importando com a lógica. Ele geralmente também fica responsável por somente uma fração do projeto.


Já o desenvolvedor tem que ter uma visão mais ampla de todo o projeto em si, ganhando mais responsabilidades, como: ajudar a elaborar o design do site, tratar do relacionamento com seus clientes, definir as tecnologias utilizadas no processo, entre varias funções

Quais tecnologias eu preciso para ser um programador Frontend?

HTML

A HTML é a linguagem para todo Frontend, visto que ela é a base para todos os sites de hoje em dia.

CSS

A Css é a linguagem responsável por dar estilo e melhorar a vizualização da página, pois ela se trata de uma linguagem de estilização.

JavaScript

O JS é a linguagem que mais exerce funções dentro dos sites atualmente, sendo uma das linguagens de programação mais utilizadas do mundo.

Preciso de um computador bom?

Não!

Muitas pessoas quando pensam na programação em geral, sempre pensam em grandes computadores super potentes que processam tudo, mas isso não é verdade.


Claro, para programas mais complexos isso é importante, porém para o nível iniciante em programação isso ta longe de ser uma realidade.

O mínimo que precisamos no inico da programação Web é de um editor de texto e um navegador. O editor de texto pode ser qualquer editor que permita o usuário salvar o arquivo com extensões .html e .css.


Você também pode baixar editores de códigos "leves", como o Notepad++ ou até mesmo o SublimeText
